# seiken3paedia
Seiken Densetsu 3 party team generator

# Tools used :  
Plates (PHP template generator)  
Bootstrap 4  
jQuery 3  
SASS  
mySQL  
MeDoo (mySQL PHP connector)  

# aspects  
Characters (sorted by class)  
Teams (by ids)  

# goal  
describe in a pinch all characteristics of a team party
